📌 Project Overview

The Data Science Salary Calculator is an interactive Excel dashboard designed to analyze salary trends across various data-related roles, countries, and employment types. This project was built using Microsoft Excel to demonstrate data analysis, dashboarding, and interactive reporting skills using real-world job salary data.

The dashboard allows users to dynamically explore salary distributions and compare compensation across different data roles including Data Analyst, Data Scientist, and Data Engineer.

This project was built as part of a hands-on data analytics learning project using real-world job postings dataset.

🎯 Project Objectives

Compare salaries across countries

Build an interactive Excel dashboard
